hive-mind

by arctype-ventures · v0.1.6 · Active
Overview
Skills
Changelog
Reviews

Cross-project AI knowledge system for Claude Code. Maintains structured summaries of repos, issues, and web sources so Claude always has full context without re-exploring codebases from scratch.

Included Skills
/hive-mind-add
Add a source with auto-summarization
/hive-mind-update
Refresh summaries against latest code
/hive-mind-pull
Update git submodules to latest
/hive-mind-idea
Create exploration and design docs
/hive-mind-summary
Generate aggregated cross-source summaries

Install

/plugin install hive-mind

Paste into Claude Code to install.
